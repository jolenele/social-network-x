# NewMe – AI-Powered Style Preview

NewMe lets you experiment with new hairstyles and colors using your own Google Photos. Powered by AI, it’s fun, easy, and privacy-friendly.

## Features
- Sign in with Google for secure access
- Import and preview your photos
- Try trending hairstyles and colors
- Save your favorite looks

## Getting Started
1. Clone the repo
2. Copy `.env.example` to `.env.local` and fill in your Google API credentials
3. Install dependencies:  
  ```bash
  npm install
  ```
4. Run the app:  
  ```bash
  npm run dev
  ```

## Tech Stack
- Next.js 16
- React 19
- Tailwind CSS
- Google Photos Picker API

## Troubleshooting
Ensure Google console and API keys are properly set up.
